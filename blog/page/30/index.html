源码安装gnuplot

2014 年9 月17 日

On the centos server, I want to install gnuplot in my local dir, here are the steps:

  1. download the source code and cd into the dir.

  2. ./configure –prefix=$HOME/local –with-readline=gnu

  3. make

  4. make install

That is all.
